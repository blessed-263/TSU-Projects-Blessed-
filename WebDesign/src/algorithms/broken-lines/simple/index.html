Broken Lines & Spline Interpolation

Algorithm Overview

This tool lets you create a broken line (polyline) by clicking points on the canvas. You can then enable spline interpolation to smoothly connect the points. This page implements a B-spline due to its:

  1. Flexibility: The degree of the B-spline curve is independent of the number of control points, allowing you to add as many points as needed without increasing complexity.
  2. Smoothness: B-splines provide smooth curves that are visually pleasing and continuous in their derivatives, making transitions between points seamless.
  3. Local control: Moving one control point only affects a small portion of the curve, so you can adjust parts of the curve without changing the entire shape.
  4. Stability: B-splines avoid unwanted oscillations and behave well even with many control points, unlike some other spline types.
  5. Efficiency: The mathematical formulation allows for fast computation and rendering, making it suitable for interactive tools like this one.

How It Works

  • Click the canvas to add points and create a broken line.
  • Click "Interpolate Spline" to draw a smooth spline through the points.
  • Click "Reset" to clear the canvas and start over.
  • Download the result as an image.
