Advanced Spline Interpolation

Algorithm Overview

Draw a broken line by clicking points on the canvas. Enable spline interpolation to create a smooth curve. For each spline interval, adjust its convexity (curvature) using the sliders below the canvas. This algorithm applies bezier curves due to:

  1. Smoothness: Cubic Bézier curves are smooth and continuously differentiable, making them ideal for vector graphics, font design, and animation paths.
  2. Adjustable Curvature: By moving the control points, designers can precisely shape the curve.
  3. Piecewise Construction: Complex shapes are often built by connecting multiple Bézier curves with continuity conditions ensuring smooth transitions.
  4. Curve Length and Optimization: Algorithms exist to approximate curve length and optimize curvature for applications like path smoothing and collision avoidance

How It Works

  • Click the canvas to add points and create a broken line.
  • Click "Interpolate Spline" to draw a smooth spline through the points.
  • Adjust the convexity slider for each interval to change the curve's shape locally.
  • Click "Reset" to clear the canvas and start over.
  • Download the result as an image.
