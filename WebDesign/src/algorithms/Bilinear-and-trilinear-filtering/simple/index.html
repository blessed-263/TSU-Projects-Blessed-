Bilinear-and-Trilinear-Filtering

Algorithm Overview

  • Bilinear filtering estimates pixel values using a weighted average of the four nearest neighbors for smooth upscaling.
  • Trilinear filtering combines multiple mipmap levels with bilinear interpolation to handle downscaling and prevent aliasing.

How to use:

  1. Upload an image using the file input
  2. Select 3 points on the left (input) canvas
  3. Select 3 corresponding points on the right (output) canvas
  4. Click "Process Image" to apply the transformation
  5. Download the result using the "Download Result" button
