Advanced Colour-Correction/Filtering

Algorithm Overview

The colour filter algorithm applies a selected colour filter to an input image by modifying the image’s pixel colour values to emphasize or alter specific colour components. This process generates a new image that visually reflects the chosen filter effect.

This algorithm will apply 6 filters

  • Red
  • Blue
  • Green
  • Yellow
  • Cyan
  • Purple

Back to Simple Version

How It Works

  • Step 1: The user uploads an image from their device.
  • Step 2: The user selects filter to be applied (Red, Blue, Green)
  • Step 3: The algorithm applies the filter and image is available for download
