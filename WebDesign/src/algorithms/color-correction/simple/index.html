Simple Colour-Correction

Algorithm Overview

The colour filter algorithm applies a selected colour filter to an input image by modifying the image’s pixel colour values to emphasize or alter specific colour components. This process generates a new image that visually reflects the chosen filter effect.

Red Filter

The red filter algorithm increases the intensity of the red channel in each pixel while reducing the green and blue channels. This makes reds more prominent and can warm the overall image tone. It works by scaling or adding to the red values and attenuating the other channels, effectively tinting the image red.

Blue Filter

The blue filter algorithm enhances the blue channel of each pixel, suppressing red and green components. This creates a cooler, bluish effect on the image. The filter adjusts pixel values to emphasize blues, which can evoke a cold or serene mood.

Green Filter

The green filter algorithm boosts the green channel intensity while reducing red and blue channels. This highlights greens and can give the image a fresh, natural look. It modifies pixel colours to emphasize green hues, often used to enhance foliage or nature scenes.
