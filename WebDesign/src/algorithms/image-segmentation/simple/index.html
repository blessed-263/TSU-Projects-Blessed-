Simple Image Segmentation

Algorithm Overview

This segmentation tool allows you to load an image and highlight geometric shapes within it. The segmentation mode identifies and outlines simple objects in images that contain them, producing a result with the segments visually highlighted.

How Image Segmentation Works in This App

This application performs basic image segmentation using built-in JavaScript and the HTML5 Canvas API. It first converts the uploaded image to grayscale, then applies the Sobel edge detection algorithm to identify edges by calculating gradients in pixel intensity. Detected edges are highlighted in red, allowing you to visually distinguish shapes and boundaries in the image. No external libraries are used; all processing is done through direct pixel manipulation on the canvas.

How It Works

  • Upload an image containing geometric shapes.
  • Select "Segmentation Mode" to process the image.
  • The algorithm analyzes the image, detects geometric shapes, and highlights their boundaries on the canvas.
  • The result is an image with the detected segments visually outlined or colored.
