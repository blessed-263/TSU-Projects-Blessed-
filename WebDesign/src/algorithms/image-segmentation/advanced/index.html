Simple Image Segmentation

Algorithm Overview

This segmentation tool allows you to load an image and highlight geometric shapes within it. The segmentation mode identifies and outlines simple geometric objects (such as circles, rectangles, and triangles) in images that contain them, producing a result with the segments visually highlighted.

Back to Simple Version

How It Works

  • Upload an image containing geometric shapes.
  • Select "Segmentation Mode" to process the image.
  • The algorithm analyzes the image, detects geometric shapes, and highlights their boundaries on the canvas.
  • The result is an image with the detected segments visually outlined or colored.
