Simple Image Retouching

Algorithm Overview

This simple retouching tool allows you to select an area on the image to be retouched. You are provided with retouching options to apply to the image

Retouching Options:

  1. Grayscale
  2. Blur
  3. Sharpen
  4. Spot Removal
  5. Vignette
  6. Dodge and Burn

How It Works

  • Upload an image.
  • Choose a retouching option.
  • Click and drag on the image to select the area to retouch.
  • Release the mouse to apply the retouch effect.
  • Download the retouched image if satisfied.
