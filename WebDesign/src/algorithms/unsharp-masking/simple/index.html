Simple Unsharp Masking

Algorithm Overview

Unsharp masking enhances the sharpness of an image by subtracting a blurred version of the image from the original, amplifying edges and fine details. This is useful for sharpening scanned photos or emphasizing texture.

How It Works

  • The user selects an image to upload.
  • The user adjusts parameters for amount, radius, and threshold.
  • The algorithm processes and sharpens the image using those values.
